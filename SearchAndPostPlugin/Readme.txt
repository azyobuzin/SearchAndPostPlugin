Search and Post plugin 1.0
作者：@azyobuzin
ライセンス：MIT License

「Google:～」や「Wikipedia:～」とツイートするとブラウザが起動して勝手に検索してくれるKrile用プラグインです。
pluginフォルダにSearchAndPostPlugin.dllをコピーすることで有効になります。

★対応している検索サービス
	・Google
	・Wikipedia
	・MSDN

	増やしたりURIを変更したい場合はソースコードを改造したり、
	新たにプラグインを作ってEntryPoint.ServiceDictionaryを弄ったりしてあげてください。
