Search and Post plugin 1.1.2
作者：@azyobuzin
ライセンス：MIT License
動作環境：Krile 2.0.8以上とそれが動くマシン

「Google:～」や「Wikipedia:～」とツイートするとブラウザが起動して勝手に検索してくれるKrile用プラグインです。
pluginsフォルダにSearchAndPostPlugin.dllをコピーすることで有効になります。

★対応している検索サービス
	・Google
	・Wikipedia
	・MSDN

	増やしたりURLを変更したい場合はソースコードを改造したり、
	新たにプラグインを作ってEntryPoint.ServiceDictionaryを弄ったりしてあげてください。
	そのうち設定画面もつけたい。。
